# Latihan LeetCode

Repo ini saya pakai khusus untuk latihan soal-soal LeetCode.
Isinya akan terus bertambah seiring saya ngerjain soal baru dan belajar pola yang beda-beda.

## Tujuan

- Nyimpen hasil latihan biar gampang dilihat lagi nanti.
- Nulis penjelasan dengan bahasa saya sendiri, supaya saya benar-benar paham.
- Ngebiasain diri buat mikir dari cara paling simpel dulu, lalu cari cara yang lebih rapi dan efisien.

## Cara Saya Nulis Catatan

Setiap soal biasanya saya tulis seperti ini:

1. Nama soal.
2. Inti masalahnya.
3. Cara saya mikir buat nyelesainnya.
4. Penjelasan singkat kenapa solusi itu jalan.
5. Kode jawaban.
6. Catatan kalau ada hal penting yang perlu diingat lagi.
